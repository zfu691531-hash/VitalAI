## F1：基础配置层（地基，所有模块依赖它）

### 目标
搭建全局配置、Axios封装、Pinia状态管理、Vue插件注册，其他模块的运行基础。

### 需要实现的文件（5个）

| 序号 | 文件 | 核心内容 |
|---|---|---|
| 1 | `vite.config.js` | 开发代理 `/api` → `http://localhost:8000` |
| 2 | `src/main.js` | 注册 ElementPlus(按需)、Pinia、Router、v-permission指令 |
| 3 | `src/api/request.js` | Axios实例：请求拦截器(token注入) + 响应拦截器(code≠200提示, 401跳转) |
| 4 | `src/stores/user.js` | 用户状态：token、userInfo、role、login()、logout() |
| 5 | `src/utils/auth.js` | token存取：getToken()、setToken()、removeToken() |

### 关键决策
- **token存储**：localStorage + Pinia双写，key为 `aistu_token`
- **请求拦截**：`Authorization: Bearer ${token}`
- **响应处理**：统一解构 `{ code, msg, data }`，code非200时 `ElMessage.error(msg)`
- **401处理**：清除token → 跳转 `/login`

### 自测验证
```bash
cd frontend
npm install
npm run dev
# 浏览器访问 http://localhost:5173
# 控制台无报错，页面正常渲染
```

| 序号 | 文件 | 核心内容 |

### 路由权限配置示例
```js
// routes.js
{
  path: '/students',
  component: () => import('@/views/student/StudentList.vue'),
  meta: { title: '学生管理', requiredRoles: ['teacher', 'admin'] }
}
```

### 关键业务逻辑
- **菜单过滤**：根据用户角色过滤左侧菜单项
- **权限守卫**：

- **面包屑**：根据路由meta自动生成

### useTable 设计
```js
export function useTable(fetchApi) {
  const loading = ref(false)
  const tableData = ref([])
  const pagination = reactive({ page: 1, pageSize: 10, total: 0 })
  
  const fetchData = async (params = {}) => {
    loading.value = true
    try {
      const res = await fetchApi({ ...pagination, ...params })
      tableData.value = res.data.list
      pagination.total = res.data.total
    } finally {
      loading.value = false
    }
  }
  
  return { loading, tableData, pagination, fetchData }
}
```

### useAiGenerate 设计
```js
export function useAiGenerate(apiFn) {
  const loading = ref(false)
  const error = ref(null)
  const result = ref('')
  
  const generate = async (params) => {
    loading.value = true
    error.value = null
    try {
      const res = await apiFn(params)
      result.value = res.data
    } catch (e) {
      error.value = e.message
    } finally {
      loading.value = false
    }
  }
  
  return { loading, error, result, generate }
}
```
