二、核心功能设计
功能1：HMM情绪状态感知——不用老人说出来就能懂
老人往往不会直接说"我今天不开心"。精神Agent通过分析对话中的语言特征（语速、词汇选择、句子长度、话题）来推断情绪状态，再用HMM建模情绪的转变序列，识别"连续低落"这种隐藏的危险状态。


功能2：记忆检索与连续性——真正的"懂你"
普通AI每次对话都是陌生人，老人每次都要重新介绍自己。精神Agent基于Hindsight四网络记忆系统，在每次对话开始前自动召回与当前话题相关的记忆，让对话有连续感，让王奶奶感觉"这个助手真的记得我说过的话"。

功能3：智慧传承——让一生的故事有地方安放
这是本项目最有温度的功能。很多老人有一辈子的故事、智慧、遗憾，但没有机会系统地记录下来。精神Agent扮演"传记作家"，引导老人讲述自己的故事，自动整理成结构化的人生档案，供家人永久珍藏，实现真正的"精神传承"。


功能4：反思评分——每天的自我觉察
图谱中"反思闭环：每天晚上进行总结：结合置信度评分"。精神Agent在每天结束时（21:00-22:00）做一个轻松的日总结，帮助王奶奶回顾今天，既是情绪整理，也是系统更新观点网络的机会。

// 情绪状态识别：从对话特征提取情绪信号
class EmotionAnalyzer {

  extractFeatures(conversation_text, audio_features) {
    return {
      // 语言特征
      sentence_avg_length: avgWordCount(conversation_text),  // 句子越短→情绪越低
      positive_word_ratio: posWordCount(conversation_text) / totalWords(conversation_text),
      self_reference_count: countWord(conversation_text, ["我","自己","我的"]),
      future_reference:    countWord(conversation_text, ["明天","以后","等到"]),
      negative_markers:    countWord(conversation_text, ["算了","没意思","不知道","随便"]),

      // 对话行为特征
      response_delay_sec:  audio_features.avg_pause_before_response,  // 回应延迟
      silence_count:       audio_features.silence_segments.length,    // 沉默次数
      topic_initiative:    countUserInitiatedTopics(conversation_text),
    }
  }

  // HMM情绪序列分析：识别"连续低落"这一隐藏状态
  detectEmotionPattern(daily_scores, window=7) {
    // daily_scores是过去7天的情绪评分序列
    const trend = linearRegression(daily_scores)  // 计算趋势斜率
    const consecutive_low = daily_scores.filter(s=>s<0.5).length

    if (trend.slope < -0.05 && consecutive_low >= 3) {
      return { state: "prolonged_low", intervention_needed: true }
    }
    if (daily_scores.slice(-1)[0] < 0.35) {
      return { state: "acute_low", intervention_needed: true }
    }
    return { state: "normal", intervention_needed: false }
  }
}

// 每日反思对话（21:00触发）
async function dailyReflection(user_id) {
  const today_emotion = await getEmotionHistory(user_id, "today")
  const today_events  = await getSignificantEvents(user_id, "today")

  // 根据今日情绪选择对话风格
  const tone = today_emotion.avg > 0.7 ? "celebratory" : // 高兴→一起庆祝
               today_emotion.avg > 0.5 ? "warm"         : // 平稳→温暖陪伴
                                         "gentle_support"  // 低落→温柔支持

  // 生成个性化的开场白（不是模板话）
  const opener = await llm.generate({
    instruction: `用${tone}的语气，基于今日事件，为王奶奶生成一句今日总结的开场白`,
    context: { events: today_events, emotion: today_emotion }
  })
  // 例："奶奶，今天和孙子通话了，听起来您开心了很多。这一天过得怎么样？"

  // 对话结束后，写入反思记录
  await memorySystem.retain({
    fact_type: "experience",
    text: `我（精神Agent）完成了与王奶奶的今日反思对话，情绪评分${today_emotion.final}，
           今日最重要的情感事件：${today_events.top_event}`,
    user_id,
  })
}
